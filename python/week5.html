مقادیر Truthy و Falsy

در زبان پایتون بجز نوع Boolean که مقادیرش می‌تونه فقط True یا False باشه، هر موقع که لازم باشه بقیه انواع داده نیز می‌تونن True یا False باشن. مواقعی که انواع داده‌ی دیگه به عنوان condition یا شرط استفاده بشن (مثلا به عنوان شرط if/elif). اینجا لیست مهمترین‌هاشون رو با چند تا مثال می‌گذارم:

  • رشته‌ی خالی: False ، در غیر این صورت True
  • لیست خالی (و غیره، مثل tuple، set و غیره): False ، در غیر این صورت True
  • عدد صفر: False ، بقیه‌ی اعداد True
  • عدد مختلط 0j : ‏False ، بقیه اعداد مختلط (مثل 5 + 3j) True
  • مقدار range خالی مثلا range(0) یا range(5, 1) : False ، در غیر این صورت True
  • نوع داده‌ی None : همیشه False

تو این مثال از while هم استفاده کردم که یه کم پایین‌تر بهش می‌رسیم


my_list = []
if my_list:
    print("Everything fine!")
else:
    print("The list is empty!")

number = 2.5
while number:
    print(number, end=" ")
    number -= 0.5

#>> The list is empty!
#>> 2.5 2.0 1.5 1.0 0.5

از bool() هم میشه استفاده کرد که مقدار True یا False اون مقدار رو بهمون برمی‌گردونه 


print(1, bool(15))
print(2, bool(""))
print(3, bool([4, 5]))
print(4, bool(None))

#>> 1 True
#>> 2 False
#>> 3 True
#>> 4 False

عملگرهای in و is

عملگر in: value in Iterable

عملگر in بررسی می‌کنه که آیا مقداری داخل یک مجموعه هست یا نه. اگر اون مقدار داخل اون Iterable (مثل یه لیست، رشته، یا هر مجموعه‌ی دیگه‌ای) باشه مقدار True رو برمی‌گردونه و در غیر این صورت False مثلا 1 in [1, 2, 3] مقدارش True هست، ولی "Jack" in ["Alice", "Ana"] مقدار False رو برمی‌گردونه. 


fruits = ["apple", "orange", "grape", "narangi"]

fruit = "strawberry"
if fruit in fruits:  # "in" operator here returns False
    print(f"{fruit} is available")
else:
    print(f"{fruit} is out of stock")

fruit = "narangi"
if fruit in fruits:  # "in" operator here returns True
    print(f"{fruit} is available")
else:
    print(f"{fruit} is out of stock")

#>> strawberry is out of stock
#>> narangi is available

عملگر is در پایتون

عملگر is در پایتون برای بررسی اینه که آبا دو متغیر به یک به یک مکان مشترک از حافظه اشاره می‌کنند یا نه. این عملگر با عملگر == متفاوته که فقط بررسی می‌کنه که آیا مقادیر دو متغیر با هم برابر هستند یا نه، در حالی که ممکنه توی دو قسمت متفاوت حافظه ذخیره شده باشن.

کاربرد این عملگر محدوده، ولی درکش مهمه. اینجا با چند تا مثال سعی می‌کنم بهتر نحوه عملکردش رو نشون بدم.

توی کدهای زیر دو تا متغیر در حالت‌های مختلف با استفاده از عملگر is با هم مقایسه می‌کنیم. توی کد پایین دو مقدار متفاوت رو توی دو متغیر ذخیره می‌کنیم که مشخصا مقدار False برمی‌گردونه: 


value1 = 15
value2 = 10

if value1 is value2:
    print("Those two values are the same object")
else:
    print("Those values are different objects")

#>> Those values are different objects

توی کد زیر، دو تا رشته مقادیر برابری دارن (هر دو abcd هستن) ولی توی دو تا مکان مختلف از حافظه ذخیره شدن. برای همین عملگر is مقدار False رو برمی‌گردونه و دستورای else اجرا میشن. 


value1 = "abcd"
value2 = "abcde"[:4]

# checking to see if they have the same value
print(value1, value2)

# checking if they are the same object:
if value1 is value2:
    print("Those two values are the same object")
else:
    print("Those values are different objects")

#>> abcd abcd
#>> Those values are different objects

اگر دو متغیر به یک آدرس از حافظه اشاره کنن، عملگر is مقدار True رو برمی‌گردونه. مثل کد زیر: 


value1 = "Alice"
value2 = value1

# checking if they are the same object:
if value1 is value2:
    print("Those two values are the same object")
else:
    print("Those values are different objects")

#>> Those two values are the same object

عمده استفاده‌ی این عملگر در شیءگرایی هست که بعدا بهش می‌رسیم.

عملگرهای and و or و not

عملگرهای منطقی and و or و not روی مقادیر boolean عملیاتی رو انجام میدن که هر کدوم رو پایین میگم

عملگر and

این عملگر بین دو تا مقدار boolean قرار می‌گیره، اگر هردو مقدار True باشن مقدار True رو برمی‌گردونه و در غیر این صورت False. یعنی هر دو مقدار باید True باشن که نتیجه هم True بشه. توی جدول پایین حالت‌های مختلفش رو نشون دادم:

and Operator
Val1 Val2 Val1 and Val2
True True True
True False False
False True False
False False False

به عنوان مثال توی کد زیر فقط وقتی شرط برقراره که هر دوی "narangi" و "orange" توی لیستمون باشن: 


fruits = ["apple", "orange", "grape", "narangi"]

if "narangi" in fruits and "orange" in fruits:
    print("you have my favorite fruits!")

#>> you have my favorite fruits!

عملگر or

این عملگر هم بین دو تا مقدار boolean قرار می‌گیره، و تا وقتی که یکی از دو مقدار True باشه مقدار True برمی‌گردونه، و فقط وقتی False میشه که هر دو مقدار False باشن. جدول پایین حالت‌های مختلف رو نشون میشده:

or Operator
Val1 Val2 Val1 or Val2
True True True
True False True
False True True
False False False

توی کد پایین، اگر نوع داده int یا float باشه، چاپ می‌کنه که این عدده. در غیر این صورت میگه که این عدد نیست. 


x = 0.5

if type(x) == int or type(x) == float:
    print(f"{x} is a number!")
else:
    print(f"{x} is not a number!")

#>> 0.5 is a number.

عملگر not

این عملگر قبل از یک مقدار boolean میاد و اون رو برعکس می‌کنه، یعنی اگر True باشه میشه False و اگر False باشه میشه True :

or Operator
value not value
True False
False True

مثلا توی کد زیر از کاربر می‌خوایم که یه اسم وارد کنه. اگر هیچی وارد نکرد و فقط دکمه Enter رو زد یه پیغام نمایش بده که متن خالیه: 


text = input("Enter a name: ")

if not text:
    print("You did not enter any text.")
else:
    print(f"You entered {text}")

حلقه‌ی while

قبلا حلقه‌ی for رو گفتیم، و کارش این بود که به تعداد مشخصی یک سری دستورای تکراری رو انجام می‌داد. نوع دیگه‌ی حلقه در پایتون حلقه‌ی while هست که دستورای داخلش رو تا وقتی که یک شرطی برقرار هست تکرار می‌کنه. نحوه‌ی نوشتن حلقه‌ی while به شکل زیره: 


while condition:
    # while statements

یعنی اول کلمه‌ی کلیدی while، بعد شرط حلقه و بعد دو نقطه (:) و دستورای داخل حلقه که قراره تکرار بشن با یه فرورفتگی (indent) زیرش نوشته میشن. دستورای حلقه تا وقتی که شرط حلقه برقراره اجرا میشن، به این صورت که شرط حلقه چک میشه، اگر True بود دستورای داخل حلقه اجرا میشه و دوباره به اول حلقه برمی‌گرده. دوباره شرط حلقه چک میشه و این عمل تکرار میشه تا وقتی که شرط حلقه False باشه.

شرط حلقه باید داخل حلقه تغییر بکنه، در غیر این صورت حلقه بی‌نهایت شکل می‌گیره یعنی هیچوقت تموم نمیشه و دستورای داخل حلقه تا وقتی که خودمون به زور برنامه رو متوقف نکنیم هی اجرا میشن. کد پایین یه نمونه از یه حلقه‌ی بی‌نهایته که بی‌نهایت بار کلمه‌ی "Armita" رو چاپ می‌کنه. 


x = 5

# infinite loop, because the condition is always True
while x < 10:
    print("Armita")

#>> Armita
#>> Armita
#>> Armita
#>> Armita
...

توی کد زیر تا وقتی که کاربر password رو درست وارد نکنه ازش درخواست password می‌کنیم: 


password = "123easyPass"

while input("Enter the password: ") != password:
    print("Password is wrong. try again!")

print("You are logged in")

توی کد بالا و توی شرط حلقه، مقداری که توی input از کاربر گرفته میشه با مقدار password مقایسه میشه و اگر برابر نبودن دستورای داخل حلقه اجرا میشن (یعنی چاپ میشه که پسورداشتباهه) اگر password درست وارد شده بود، شرط داخل حلقه False میشه و از حلقه میاد بیرون و چاپ میشه که کاربر لاگین شده.

حالا فرض کنید یک سیستم ثبت نام درست کردیم و می‌خوایم تا وقتی که کاربر username و password رو درست وارد نکرده ازش درخواست کنیم که اونا رو وارد کنه. این کار رو می‌تونیم به شکل زیر هم انجام بدیم: 


correct_username = "Armita"
correct_password = "123easyPass"

username = input("Enter the username: ")
password = input("Enter the password: ")
while username != correct_username and password != correct_password:
    print("username or password is wrong. try again!")
    # here we ask for username and password again
    username = input("Enter the username: ")
    password = input("Enter the password: ")

print("You are logged in")

دستورات break و continue

از دستورای break و continue فقط داخل حلقه استفاده میشه.

دستور break

در هر کجای حلقه اگر به دستور break برسیم، مستقیم از حلقه خارج میشیم و دیگه به اول حلقه بر نمی‌گردیم. مثلا اگر لیستی از اعداد داریم و بخوایم تا وقتی که عددها کوچیکتر هستن اونا رو چاپ کنیم و اگر به عدد 10 یا بزرگ‌تر رسیدیم بقیه رو چاپ نکنیم، می‌تونیم از دستور break توی حلقه‌ی for استفاده کنیم: 


numbers = [7, 2, 8, 4, 11, 19, 3, 18, 6, 12]

for num in numbers:
    if num >= 10:
        break
    print(num)
print("break statement finished the loop early!")

#>> 7
#>> 2
#>> 8
#>> 4
#>> break statement finished the loop early!

توی کد بالا اعداد 7 و 2 و 8 و 4 که کوچکتر از 10 هستن چاپ شدن، عدد بعدی 11 هست که باعث شد شرط if برقرار باشه و دستور break اجرا بشه. با اجرا شدن دستور break از حلقه خارج شدیم و دستور بعدی که بیرون حلقه هست اجرا میشه.

دستور continue

توی یک حلقه، اگر به دستور continue برسیم، بقیه دستورای داخل حلقه نادیده گرفته میشن و به اول حلقه برمی‌گردیم. توی حلقه‌ی while وقتی به دستور continue برسیم به اول حلقه برمی‌گرده شرط حلقه دوباره چک میشه. توی حلقه‌ی for هم اگر به دستور continue برسیم، به اول حلقه برمی‌گردیم و مقدار بعدی اون لیست یا ... رو استفاده می‌کنیم.

مثلا توی کد بالا، فرض کنید می‌خواستیم تمام اعدادی که کوچیکتر از 10 هستن رو چاپ کنیم به جای اینکه از حلقه خارج شیم. برای اینکار می‌توین از دستور continue به شکل زیر استفاده کنیم: 


numbers = [7, 2, 8, 4, 11, 19, 3, 18, 6, 12]

for num in numbers:
    if num >= 10:
        continue
    print(num)
print("All the numbers less than 10 are printed")

#>> 7
#>> 2
#>> 8
#>> 4
#>> 3
#>> 6
#>> break statement finished the loop early!

توی کد بالا، هر موقع متغیر حلقه 10 یا بزرگتر بوده، مستقیم به اول حلقه برگشتیم و عدد رو چاپ نکردیم.
