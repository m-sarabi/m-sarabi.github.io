نوع داده‌ی tuple

داده‌ی tuple مثل list یک Iterable هست که می‌تونه انواع داده‌ی دیگه رو در خودش نگهداری کنه. برای ساختن داده‌ی tuple از پرانتز باز و بسته () استفاده می‌کنیم.

بزرگترین تفاوت tuple با list اینه که tuple غیرقابل تغییر هست، یعنی وقتی ساخته شد دیگه نمیشه عضوی رو بهش اضافه کرد، ازش کم کرد و یا تغییر داد. پس متدهایی مثل append و remove و pop و غیره دیگه برای tuple وجود ندارن. 


my_tuple = (1, 2, "a", False)
print(my_tuple[2:])

#>> ('a', False)

تفاوت دیگه‌ای که نوع tuple با list اینه که tuple فضای کمتری از حافظه رو اشغال می‌کنه. و همین باعث میشه که تو دیتای زیاد سرعت دسترسی به دیتا هم مقداری بیشتر بشه. مثلا توی کد زیر می‌بینید که لیست 72 بایت از حافظه رو اشغال کرده، ولی همون دیتا از نوع تاپل 48 بایت رو اشغال کرده. 


my_list = ["a", "b", "c"]
my_tuple = ("a", "b", "c")

print(my_list.__sizeof__())
print(my_tuple.__sizeof__())

#>> 72
#>> 48

نوع داده‌ی dict دیکشنری

نوع داده‌ی dict یا دیکشنری توی پایتون به صورت key و value یا کلید و مقدار هستن. کلید و مقدار رو باید با یک علامت دو نقطه : از هم جدا کرد، و جفت کلید و مقدارها رو هم مثل لیست یا tuple با یک کاما از هم جدا می‌کنیم. مقدار value از هر نوع داده‌ای می‌تونه باشه، ولی key باید قابل هش (hashable) باشه. معمولا انواع داده‌ای که قابل تغییر نیستن hashable هستن (مثل string و عدد و تاپل و غیره)، ولی انواع داده‌ای که قابل تغییر هستن (مثل list و دیکشنری و set) قابل هش نیستن و نمیشه به عنوان key استفاده بشه.

البته یک تاپل که که یک آیتم غیر قابل هش (مثلا یک لیست) داخلش باشه هم خودش قابل هش نیست. تو مثال‌های زیر بعضی از انواع key های مختلف رو می‌بینید. 


dictionary = {
    "hello": 1,
    3.14: 2,
    (0, 0, "a"): 3,
    False: 4,
    None: 5,
    range(2, 40, 3): 6
}

برای دسترسی به value یک key داخل دیکشنری، key رو داخل براکت می‌نویسیم (مثل لیست، با این تفاوت که بجای ایندکس از key استفاده می‌کنیم: 


dictionary = {
    "color": "blue",
    "age": 98,
    "weight": 350,
}

print(dictionary["color"])

#>> blue

بعضی متدهای دیکشنری‌ها به قرار زیر هستند:

dict.keys() و dict.values() : تمام key ها یا value های دیکشنری رو به صورت یک Iterable برمی‌گردونه.

dict.items() : تمام کلید و مقدارهای دیکشنری رو به صورت یک Iterable برمی‌گردونه (هر کلید و مقدار داخل یک tuple هستن).

روی دیکشنری هم مثل بقیه‌ی iterable ها می‌تونیم حلقه بزنیم. اگر بخوایم تنها روی کلیدها حلقه بزنیم به یکی از روش‌های زیر می‌تونیم این کار رو بکنیم: 


dictionary = {
    "color": "blue",
    "age": 98,
    "weight": 350,
}

# method 1
for i in dictionary:
    print(f"({i}, {dictionary[i]})", end=" ")

print()

# method 2, same as above
for i in dictionary.keys():
    print(f"({i}, {dictionary[i]})", end=" ")

#>> (color, blue) (age, 98) (weight, 350)
#>> (color, blue) (age, 98) (weight, 350)

اگر بخوایم تنها روی value های دیکشنری حلقه بزنیم می‌تونیم از متد dict.values() استفاده کنیم: 


dictionary = {
    "color": "blue",
    "age": 98,
    "weight": 350,
}

# method 1
for i in dictionary.values():
    print(i)

#>> blue
#>> 98
#>> 350

و یا اگر بخوایم روی هر دو key و value حلقه بزنیم از متد dict.items() می‌تونیم استفاده کنیم: 


dictionary = {
    "color": "blue",
    "age": 98,
    "weight": 350,
}

# method 1
for i, j in dictionary.items():
    print(f"({i}, {j})", end=" ")

#>> (color, blue) (age, 98) (weight, 350)
